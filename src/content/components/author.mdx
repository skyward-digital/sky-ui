## Usage

```jsx
import { Author } from "#/ui/components/Author"
```

```jsx
<Author
  avatar={{
    src: "https://a.storyblok.com/f/180053/1200x1600/bbce43daaf/dan-spratling-profile-picture.jpg",
    alt: "Dan Spratling",
  }}
  name="Dan Spratling"
/>
```

## Link

By passing a href prop the author will become a link, ideal for linking to author profiles.

<CodePreview client:load>
  <Author
    avatar={{
      src: "https://a.storyblok.com/f/180053/1200x1600/bbce43daaf/dan-spratling-profile-picture.jpg",
      alt: "Dan Spratling",
    }}
    name="Dan Spratling"
    href="#dan-spratling"

/>

  <div slot="code">
    ```jsx
    <Author
      avatar={{
        src: "https://a.storyblok.com/f/180053/1200x1600/bbce43daaf/dan-spratling-profile-picture.jpg",
        alt: "Dan Spratling",
      }}
      name="Dan Spratling"
      href="/team/dan-spratling"
    />
    ```
  </div>
</CodePreview>
