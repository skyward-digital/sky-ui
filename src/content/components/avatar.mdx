## Usage

```jsx
import { Avatar } from '@/ui/components/Avatar'
```

```jsx
<Avatar
  src='https://a.storyblok.com/f/180053/1200x1600/bbce43daaf/dan-spratling-profile-picture.jpg'
  alt='Dan Spratling'
  fallback='Dan Spratling'
/>
```

## Image & fallback

The avatar component has a fallback option that will display the user's initials if no image is provided.

<div className='not-prose border rounded-lg p-6 flex items-center justify-center gap-2'>
  <Avatar
    src='https://a.storyblok.com/f/180053/1200x1600/bbce43daaf/dan-spratling-profile-picture.jpg'
    alt='Dan Spratling'
    fallback='Dan Spratling'
  />
  <Avatar fallback='Dan Spratling' />
</div>

Fallbacks are based on the fallback prop, automatically caulculating the best initials to display.

<div className='not-prose border rounded-lg p-6 flex items-center justify-center gap-2'>
  <Avatar fallback='Dan Spratling' />
  <Avatar fallback='Dan' />
  <Avatar fallback='DS' />
</div>

## Customisation

### Classes

You can customize avatars by passing a `className` prop. If this clashes with the default styles, the `className` will take precedence.

<div className='not-prose border rounded-lg p-6 flex items-center justify-center gap-2'>
  <Avatar
    className='rounded-lg rounded-tl-none'
    src='https://a.storyblok.com/f/180053/1200x1600/bbce43daaf/dan-spratling-profile-picture.jpg'
    alt='Dan Spratling'
    fallback='Dan Spratling'
  />
  <Avatar className='bg-orange-500 text-white' fallback='Dan Spratling' />
</div>
